## Common Keymap

- command+tab
- command+~
- command+control+f - Window Maximum
- command+shift+3   - Print all screen
- command+shift+4   - Print area screen

## Terminal

- ctrl+d - close
- ctrl+c - terminal
- ctrl+r - history search
- ctrl+l - clear

- ctrl+b - left
- ctrl+f - right
- ctrl+p - up
- ctrl+n - down
- ctrl+a - line first
- ctrl+e - line end

- ctrl+k - delete to end
- ctrl+u - delete to first
- ctrl+d - delete char
- ctrl+w - delete word
- ctrl+y - paste from ctrl+u，ctrl+d，ctrl+w
