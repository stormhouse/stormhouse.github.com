9.12 垂直居中对齐(Middle Aligned)

  • 内联内容垂直居中对齐line-height == height
  • 对设定尺寸的绝对定位元素垂直居中对齐{position:absolute;height:xx;margin-top:auto;margin-bottom:auto;top:0;bottom:0;}
  • 对拉伸的绝对定位元素垂直居中对齐{position:absolute;height:xx;margin-top:xx;margin-bottom:xx;top:0;bottom:0;}
INLINE
Sized Absolute
Can't middle-align a static element or a shrinkwrapped element.
